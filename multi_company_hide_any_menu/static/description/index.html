Multi Company Hide Any Menu For Any User


If you want to hide unwanted menu, sub-menu for particular user depend on selected company for that user then just install this app.

Suppose User want to hide Sales App in Company A but should be visible in Company B, Purchase App visible in Company A but should be hidden in Company B. So user can avoid selling or buying in a wrong company. There are too many such type of scenario in such situation this app is very useful

Key features:
  • Easy To Use.
  • Hide Any Menu, Sub-Menu.
  • Setting Can Applied For Administrator Too.
  • Company Wise Configuration.
  • Same Menu Can Accessible From One Company But Hide In Another Company.
  • Set Access Right, So Only Specific User Will Change/Set Hide Menu Setting.
To enable import functionality, you have to select Multi Company Hide Any Menu access from Settings --> Users & Companies --> Ssers --> Assess Right --> Technical Settings.
Suppose You Want To Hide Some Menus Like : Main Menu Project, Purchases and some sub menu like Translations From Settings, Email From Settings --> Techinical.
Suppose mentioned menus should invisible for Admin and Demo User for company Your Company and will visible into other company.
Go to Settings --> Companies --> Companies --> select company Your Company --> Hide Menu Page --> Add line like mentioned menus and users.
Logged in as a Administrator user current company is Your Company
configured menus and sub menus are invisible for user Administrator and company Your Company
Logged in as a Demo User user current company is Your Company
configured menus and sub menus are invisible for user Demo User and company Your Company
Logged in as a Administrator user current company is Company_1(other company)
All menus and sub menus are visible for user Demo User because Company_1 is not set for any hide menu configuration
Logged in as a Demo User user current company is company_1(other company)
All menus and sub menus are visible for user Demo User because Company_1 is not set for any hide menu configuration
